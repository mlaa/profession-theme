#Tasks for Rolling out Archive and Categories Pages: 

 * [ ] Dashboard → Pages → Add New: enter: 
    - title: Categories
    - template: Categories
    - click “Publish” 
 * [ ] Dashboard → Pages → Add New: enter: 
    - title: Archive
    - template: Archive
    - click “Publish” 
 * [ ] Dashboard → Appearance → Menus →  Pages section: 
    - check “Categories,” 
    - click “Add to Menu,” 
    - check “Archive” 
    - click “Add to Menu,” 
    - reorder pages to read (from top to bottom): Home, About, Categories, Archive, Contact. 
    - click “Save Menu.”
